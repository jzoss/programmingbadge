# ⚖️ Requirement 4: Intellectual Property

## Understanding Software Rights & Licenses

---

# 📋 What This Requirement Asks

**(a)** Explain the **four types of intellectual property** used to protect computer programs.

**(b)** Describe the difference between **licensing and owning** software.

**(c)** Describe the differences between **freeware, open source, and commercial** software, and why it's important to **respect the terms of use** of each.

---

# 🤔 What is Intellectual Property?

**Intellectual Property (IP)** = creations of the mind that are protected by law.

Just like you can own a bicycle or a backpack, people can **own ideas and creations:**
- A song 🎵
- A book 📖
- An invention 💡
- **A computer program** 💻

> **Why does this matter?** Programmers spend months or years writing software. IP laws make sure they get credit and can earn a living from their work.

---

# 🛡️ The Four Types of IP

| Type | Symbol | Protects | Duration |
|------|--------|----------|----------|
| **Copyright** | © | Creative works (code, art, music) | Life + 70 years |
| **Patent** | 📜 | Inventions & unique processes | 20 years |
| **Trade Secret** | 🤫 | Confidential business info | As long as it's secret |
| **Trademark** | ™ / ® | Brand names, logos, slogans | Forever (if renewed) |

Let's look at each one and how it applies to software...

---

<!-- _class: compact -->

# © Copyright

### What it protects:
The **actual code** you write — the specific words and structure.

### Software examples:
- The source code of a program
- The design of a user interface
- Documentation and manuals

### Key facts:
- ✅ **Automatic** — your code is copyrighted the moment you write it!
- ✅ No registration needed (but registering helps in court)
- ❌ Does NOT protect the **idea** — only the specific way you wrote it
- Someone else can write their OWN code to do the same thing

> **Analogy:** Two people can both write a recipe for chocolate chip cookies 🍪 — copyright protects the SPECIFIC recipe, not the idea of cookies.

---

<!-- _class: compact -->

# 📜 Patent

### What it protects:
A **new invention or process** — the unique WAY something works.

### Software examples:
- Amazon's "1-Click" purchase button (famously patented!)
- Google's PageRank search algorithm
- Apple's "slide to unlock" gesture

### Key facts:
- ✅ Protects the **idea/method**, not just the code
- ❌ Must be **applied for** and approved (expensive!)
- ❌ Only lasts **20 years**
- ❌ Must be a **new, non-obvious** invention

> **Analogy:** A patent on a new type of engine ⚙️ means nobody else can build that type of engine, even if they design it differently.

---

<!-- _class: compact -->

# 🤫 Trade Secret

### What it protects:
**Confidential information** that gives a company a competitive advantage.

### Software examples:
- Google's exact search ranking algorithm
- Coca-Cola's formula (non-software example everyone knows!)
- Netflix's recommendation algorithm
- A company's proprietary data processing methods

### Key facts:
- ✅ Lasts **forever** — as long as it stays secret!
- ✅ No registration needed
- ❌ If someone figures it out on their own, you **can't stop them**
- ❌ If it leaks, the protection is **gone**

> **Analogy:** A secret family recipe 🤫 — it's valuable BECAUSE nobody else knows it.

---

<!-- _class: compact -->

# ™ Trademark

### What it protects:
**Brand identity** — names, logos, and symbols that identify a product.

### Software examples:
- The **Apple logo** 🍎
- The word **"Windows"** (for Microsoft's OS)
- The **Android robot** mascot 🤖
- **"PlayStation"** name and logo

### Key facts:
- ✅ Lasts **forever** (if renewed and used)
- ✅ Prevents others from using confusingly similar names/logos
- ❌ Only protects the **brand**, not the product itself
- ™ = unregistered, ® = registered with the government

> **Analogy:** Nobody else can open a burger place called "McDonald's" 🍔 with golden arches — that's trademarked!

---

# 🏠 Part B: Licensing vs. Owning

### What's the difference?

| | Owning | Licensing |
|--|--------|-----------|
| **What?** | You actually own a copy | You have permission to use it |
| **Can sell it?** | Yes ✅ | No ❌ |
| **Can modify it?** | Usually yes ✅ | Usually no ❌ |
| **Can copy it?** | Limited ⚠️ | No ❌ |
| **Expires?** | No | Can expire |
| **Example** | Buying a physical book | Netflix subscription |

---

# 🎬 Licensing: The Netflix Analogy

This is the easiest way to understand it:

### 📀 Owning (buying a DVD):
- You **own** the disc
- You can watch it whenever you want
- You can lend it to a friend
- You can sell it at a garage sale
- If the store closes, you still have your DVD

---

# 🎬 Licensing: The Netflix Analogy (cont.)

### 📱 Licensing (Netflix subscription):
- You have **permission** to watch while you pay
- You can't download and keep the movies forever
- You can't share your account with everyone
- Netflix can **remove** a movie anytime
- If your subscription ends, you lose access

> **Most software today is LICENSED, not owned!** When you "buy" an app, you usually buy a LICENSE to use it.

---

# 💻 Software Licensing Examples

| Software | Do You Own It? | What You Get |
|----------|---------------|-------------|
| Microsoft 365 | ❌ Licensed | Access while you pay monthly |
| Minecraft (Java) | ✅ Own a license | Permanent license to play |
| Fortnite | ❌ Licensed (free-to-play) | Access that could be revoked |
| Photoshop | ❌ Licensed (subscription) | Access while you pay |
| iPhone apps | ❌ Licensed | Apple can remove them |

### Read the Fine Print!
When you click **"I Agree"** to Terms of Service, you're agreeing to a **license**, not purchasing ownership!

---

# 🆓 Part C: Types of Software

### Three Main Categories:

```
Software
├── 💰 Commercial Software     — Pay to use, code is secret
├── 🆓 Freeware               — Free to use, code is usually secret
└── 🔓 Open Source Software    — Free to use, code is open to everyone
```

Let's explore each one...

---

<!-- _class: compact -->

# 💰 Commercial Software

### What it is:
Software you **pay money** to use. The source code is **private**.

### Examples:
| Software | Price | What You Get |
|----------|-------|-------------|
| Microsoft Office | ~$100/year | Word, Excel, PowerPoint |
| Adobe Photoshop | ~$23/month | Photo editing |
| Final Cut Pro | $300 (one-time) | Video editing |

### Rules:
- ❌ Cannot see or modify the source code
- ❌ Cannot share copies with friends
- ❌ Cannot reverse-engineer it
- ✅ Usually comes with customer support
- ✅ Regular updates included

---

<!-- _class: compact -->

# 🆓 Freeware

### What it is:
Software that is **free to use**, but the source code is usually **not available**.

### Examples:
| Software | What It Does |
|----------|-------------|
| Adobe Acrobat Reader | View PDF files |
| Discord | Chat & voice for gamers |
| Zoom (free tier) | Video calls |
| Spotify (free tier) | Music streaming with ads |

### Rules:
- ✅ Free to download and use
- ❌ Usually cannot see the source code
- ❌ Cannot modify it
- ⚠️ Might have ads or limited features
- ⚠️ Company can stop offering it anytime

---

<!-- _class: compact -->

# 🔓 Open Source Software

### What it is:
Software where the **source code is available to everyone**. You can see it, modify it, and share it.

### Examples:
| Software | What It Does |
|----------|-------------|
| Linux | Operating system (runs most web servers!) |
| Firefox | Web browser |
| VLC Media Player | Plays any video/audio file |
| Python | Programming language |
| Android | Mobile operating system |
| Scratch | Programming for kids! |

### Rules:
- ✅ Free to use
- ✅ Source code is public — you can read it and learn from it!
- ✅ You can modify it for your own use
- ✅ Community-driven — anyone can contribute improvements
- ⚠️ Must follow the specific open source license terms

---

# 📊 Side-by-Side Comparison

| Feature | Commercial 💰 | Freeware 🆓 | Open Source 🔓 |
|---------|--------------|-----------|---------------|
| **Cost** | Paid | Free | Free |
| **Source code** | Private | Usually private | Public |
| **Modify it?** | No ❌ | No ❌ | Yes ✅ |
| **Support** | Paid support | Limited | Community |
| **Examples** | Photoshop, Office | Discord, Zoom | Linux, Firefox |

---

# 🤝 Why Respect the Terms?

### It's about being a good Scout! ⚜️

**Ethical reasons:**
- Developers worked hard to create that software
- Stealing software is just like stealing from a store
- The Scout Law says a Scout is **Trustworthy** and **Obedient**

**Legal reasons:**
- Software piracy (illegal copying) can result in **fines up to $250,000** 😱
- Companies CAN track illegal copies

---

# 🤝 Why Respect the Terms? (cont.)

**Practical reasons:**
- Pirated software often contains **viruses and malware** 🦠
- No updates, no security patches
- No customer support when things break

---

<!-- _class: compact -->

# 🏴‍☠️ Don't Be a Software Pirate!

### Things that are **NOT OK**:
- ❌ Downloading paid software from unofficial sites for free
- ❌ Sharing your software license key with friends
- ❌ Using one license on more computers than allowed
- ❌ Copying someone's open source code without following their license
- ❌ Claiming someone else's code as your own

### Things that are **OK**:
- ✅ Using free and open source alternatives
- ✅ Using free tiers of commercial software
- ✅ Buying proper licenses
- ✅ Using open source code AND following the license
- ✅ Learning from others' code and writing your own version
