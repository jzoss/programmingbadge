# 📜 Requirement 2: History

## The Evolution of Programming Languages

---

# 📋 What This Requirement Asks

> Discuss with your counselor the **history of programming** and the **evolution of programming languages**, including at least **three milestones** related to the advancement or development of programming over time.

Let's travel through time! ⏰

---

# 🕰️ Before Computers: The Idea of Programming

### 1800s — It All Started with Looms!

- **1801:** Joseph Marie Jacquard invented a loom that used **punch cards** to weave patterns automatically
- This was the first machine controlled by "instructions" — a kind of program!

### Ada Lovelace (1815–1852) 👩‍💻
- Worked with Charles Babbage on his **Analytical Engine** (a mechanical computer design)
- Wrote the **first computer program** — an algorithm to calculate a sequence of numbers
- She's considered the **world's first programmer!**

> Fun Fact: There's a programming language named "Ada" in her honor!

---

# 🏗️ The 1940s–1950s: First Computers & Machine Code

### First-Generation Programming: Machine Code
- **1940s:** First electronic computers (ENIAC, 1945) were programmed by **physically rewiring** cables and flipping switches! 😱
- Then came **machine code** — programming in pure 1s and 0s:
  ```
  10110000 01100001
  ```
- Only experts could do this — it was incredibly hard and error-prone

### ⭐ Milestone #1: Assembly Language (~1949)
- Used short **mnemonics** instead of binary:
  ```
  MOV AL, 61h    (instead of 10110000 01100001)
  ```
- Still close to the hardware, but **much** easier to read!

---

# 🚀 The 1950s: High-Level Languages Are Born

### ⭐ Milestone #2: FORTRAN (1957)

- Created by **John Backus** at IBM
- **First high-level programming language** — you could write code in something closer to English/math!
- Designed for **scientific and engineering** calculations
- Example:
  ```fortran
  PROGRAM HELLO
      PRINT *, 'Hello, World!'
  END PROGRAM HELLO
  ```

### COBOL (1959)
- Created by **Grace Hopper** (a Navy Admiral! ⚓)
- Designed for **business** applications
- Banks and governments STILL use COBOL today!

> Grace Hopper also found one of the first actual computer "bugs" — a real moth stuck in a computer! 🦗

---

# 🔧 The 1960s–1970s: Building Blocks

### BASIC (1964)
- Designed at Dartmouth College to teach **beginners**
- Made programming accessible to students for the first time
- Microsoft was founded when Bill Gates & Paul Allen wrote a BASIC interpreter!

### ⭐ Milestone #3: C Language (1972)
- Created by **Dennis Ritchie** at Bell Labs
- Used to build the **UNIX operating system**
- Hugely influential — most modern languages are inspired by C
- Example:
  ```c
  #include <stdio.h>
  int main() {
      printf("Hello, World!\n");
      return 0;
  }
  ```
- **macOS, Windows, Linux** — all built with C and its descendants!

---

# 🌐 The 1980s–1990s: OOP & the Web

### C++ (1983)
- Added **Object-Oriented Programming** (OOP) to C
- Used for video games (Unreal Engine), browsers, and operating systems

### Python (1991) 🐍
- Created by **Guido van Rossum**
- Designed to be **easy to read** — almost like English!
  ```python
  print("Hello, World!")
  ```

### ⭐ Milestone #4: Java (1995) ☕
- **"Write once, run anywhere"** — works on any device with a Java Virtual Machine
- Became the language for Android apps and big enterprise systems

---

# 🌍 The 1990s: The Web Changes Everything

### ⭐ Milestone #5: JavaScript (1995) 🌐

- Created by **Brendan Eich** in just **10 days!**
- Made web pages **interactive** — buttons, animations, games
- Today it's the **#1 most-used language** in the world
- Example:
  ```javascript
  alert("Hello, World!");
  ```

### HTML & CSS (not programming languages, but important!)
- **HTML** = structure of web pages (the skeleton 🦴)
- **CSS** = styling (the clothes 👕)
- **JavaScript** = behavior (the brain 🧠)

---

# 📱 The 2000s–Today: Mobile & Beyond

### Swift (2014) 🍎
- Apple's language for iPhone and iPad apps
- Replaced Objective-C — much easier to learn

### Scratch (2007) 🧩
- Created at MIT for **kids and beginners**
- Visual, drag-and-drop blocks instead of typing code
- Over **100 million** projects created!

### Modern Trends:
- 🤖 **AI & Machine Learning** — Python leads the way
- ☁️ **Cloud Computing** — code runs on servers across the world
- 📱 **Mobile apps** — Swift (iOS), Kotlin (Android)
- 🎮 **Game development** — C#, C++, Lua

---

# 📊 Timeline Summary

| Year | Milestone | Why It Matters |
|------|-----------|----------------|
| 1843 | Ada Lovelace's algorithm | First program ever written |
| ~1949 | Assembly Language | Human-readable machine code |
| 1957 | **FORTRAN** | First high-level language |
| 1959 | COBOL (Grace Hopper) | Business computing |
| 1972 | **C Language** | Foundation for modern OS & languages |
| 1991 | Python | Easy-to-read, beginner-friendly |
| 1995 | **Java** | Write once, run anywhere |
| 1995 | **JavaScript** | Made the web interactive |
| 2007 | Scratch | Programming for everyone |

# 🤔 Fun Facts to Remember

- 🦗 The term **"bug"** came from a real moth found in a computer (1947)
- 💡 JavaScript was created in **10 days** — and now runs most of the internet!
- 🏦 **95% of ATM transactions** still use COBOL code from the 1960s
- 🐍 Python is named after **Monty Python** (the comedy group), not the snake!
- 👩‍💻 Ada Lovelace wrote the first program **100 years before** the first electronic computer was built
- 🧩 Scratch has been used in **every country on Earth**
