# 🎯 What is Programming?

**Programming** = giving a computer step-by-step instructions to solve a problem.

Think of it like writing a recipe 🍳:
- **Ingredients** = Input (data)
- **Steps** = Code (instructions)
- **Finished dish** = Output (result)

Computers are FAST but LITERAL — they do exactly what you tell them!

---

# 🧑‍💻 What Does a Programmer Do?

- **Designs** solutions to problems
- **Writes** code in a programming language
- **Tests** to make sure it works
- **Debugs** when things go wrong (and they will! 🐛)
- **Collaborates** with other programmers

> "Everybody should learn to program a computer, because it teaches you how to think."
> — Steve Jobs
