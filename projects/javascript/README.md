# 🌐 Project 5b: Adventure Quiz Game (JavaScript)

## What This Program Does

A text-based adventure game that runs in the browser! You make choices, and the story changes based on your decisions. At the end, you get a score.

**Requirements met:**
- ✅ Takes **input** from the user (clicking choice buttons)
- ✅ Performs **computations** (tracks score, counts choices)
- ✅ Makes **decisions** (if/else branching based on player choices)
- ✅ Produces **output** (story text, score, win/lose message)

## Setup

No installation needed! Just:
1. Open `index.html` in any web browser
   ```bash
   open index.html    # Mac
   ```
   Or just double-click the `index.html` file!

## How to Work on This

1. **Play through the game** — try different choices!
2. **Read `adventure_game.js`** — understand the story structure

5. **Show your counselor** — explain how input → computation → decision → output works

## How the Code is Organized

- `index.html` — The web page (structure)
- `adventure_game.js` — The game logic (behavior)
- The HTML creates the visual layout (text area, buttons)
- The JavaScript handles all the logic (story, choices, score)
