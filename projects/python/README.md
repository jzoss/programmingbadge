# 🐍 Project 5a: Temperature Converter (Python)

## What This Program Does

Converts temperatures between Fahrenheit and Celsius, then gives you a fun message based on the result!

**Requirements met:**
- ✅ Takes **input** from the user (temperature + unit)
- ✅ Performs **computations** (conversion formula)
- ✅ Makes **decisions** (if/elif/else for messages)
- ✅ Produces **output** (converted temperature + message)

## Setup

1. Make sure **Python 3** is installed:
   ```bash
   python3 --version
   ```
   If not installed, download from [python.org](https://python.org)

2. Run the program:
   ```bash
   python3 temperature_converter.py
   ```

## Formulas

- **Fahrenheit to Celsius:** `C = (F - 32) × 5/9`
- **Celsius to Fahrenheit:** `F = C × 9/5 + 32`
- **Celsius to Kelvin:** `K = C + 273.15`
